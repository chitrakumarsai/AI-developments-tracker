# Slice 2 (Pause/resume ¬∑ Re-weight ¬∑ Archive ¬∑ Edit)

> Continues on the same branch/PR #12 (approved). Prefs (checkpoint 2026-07-08): **inline
> per-row controls**. Same tier + **security-reviewer** (owner data via service-role). Builds
> directly on Slice 1's `lib/sources/persist.ts`, routes, and `SourceCatalog`.

## Scope
- **Pause/resume**: `active ‚Üî paused` (paused = kept, skipped by scheduled/`due` ingestion).
- **Re-weight**: set `priority` (integer stepper, clamp `0..100`).
- **Archive / restore**: soft-delete via `status='archived'` (restore ‚Üí `active`). Archived rows
  recede + sort last; still listed (no separate view this slice).
- **Edit metadata**: `name`, `category`, `tags` only. **`url` stays immutable** ‚Äî editing it would
  require re-running SSRF/feed validation; out of scope, keeps the mutation SSRF-free.

## Architecture
```

  setSourcePriority(id, priority)          // clamp 0..100

  listSourcesWithCounts()                  // + partition archived to the bottom
app/api/sources/[id]/route.ts (NEW ‚Äî PATCH)
  requireOwner ‚Üí rate-limit("candidates") ‚Üí Zod partial body ‚Üí dispatch to the

## PATCH contract

(same string bounds as POST). uuid-validate `[id]`. 400 empty/invalid, 403 non-owner, 429 limited,
500 redacted-in-prod (mirror the POST fix).
