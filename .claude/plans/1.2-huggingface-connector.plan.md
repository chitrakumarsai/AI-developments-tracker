## Key insight
`api/router.ts` already reserved the Hugging Face slot. This slice is a new **provider** under the existing `api` type — no new registry entry, no new architecture. One connector serves both HF endpoints, dispatched by URL **path** (`/api/daily_papers` vs `/api/models`), mirroring how the router dispatches by host.
