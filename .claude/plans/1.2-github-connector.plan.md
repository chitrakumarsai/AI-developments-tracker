## Build order (test-first)

### 1. Env — `GITHUB_TOKEN`
- Add `GITHUB_TOKEN=` to `.env.example` (+ note) and `.env.local`. Server-only (never `NEXT_PUBLIC_*`).

### 2. GitHub connector — `web/lib/ingestion/api/github.ts`  *(TDD)*
- `parseGithubSearch(json, source): IngestionResult` — pure; maps `items[]` (repos) → `NormalizedItem` per the PRD table; `sanitizeText(description)`; `sanitizeUrl(html_url)`; cap 50; skip + warn on missing name/url.
- `githubConnector: Connector` — build query URL from `source.url` (topic query) + injected rolling `pushed:>=<today-30d>` + `sort=stars&order=desc&per_page=50`; `fetch` with headers `Authorization: Bearer <GITHUB_TOKEN>` (omit if unset + push a warning), `Accept: application/vnd.github+json`, `User-Agent: AIChronicles/0.1`. Non-200/403 → warning (no throw); reuse the SSRF guard (extract shared helper or keep host allowlist — api.github.com passes).
- Tests vs saved fixture `__fixtures__/github-search.json`: mapping, sanitize, missing-token warning, non-200 warning, fetch-reject warning.

### 3. API host router — `web/lib/ingestion/api/router.ts`  *(TDD)*
- `apiConnector: Connector` — parse `source.url` host: `api.github.com` → `githubConnector`; else `{ items:[], warnings:["no API connector for host <h>"] }`. (HF host next slice.)
- Registry: `api: apiConnector`. Update `registry.test.ts` (`getConnector('api')` returns the router).

### 4. Seed — `supabase/seed.sql`
- One row: name `GitHub — Notable AI repos`, category `GitHub Repositories`, `ingestion_type='api'`, `url` = `https://api.github.com/search/repositories?q=topic:machine-learning+topic:llm+topic:agents` (topics editable; connector appends date/sort/per_page), tags `['github','repos']`.

### 5. Project skill — `.claude/skills/ingestion-connector/SKILL.md`
- Capture the pattern (Connector contract → registry dispatch by ingestion_type → RSS: one connector / API: host router → sanitize + link-first + SSRF → fixture-based vitest). So HF + future connectors follow it. (CLAUDE.md §16.2.)

### 6. Live verify (needs `GITHUB_TOKEN`)
- Seed + `POST /api/ingest/run?source=<gh id>` → real repos ingested, links to github.com; re-run dedupes to 0.
